# Ondo Real Estate — Consumer Website (OndoREui)

> Next.js 15 App Router public site for buyers, sellers, investors, tenants, and property owners.
> Live: https://ondorealestate.com
> Local dev: `npm run dev` → http://localhost:3000

## What This Application Does

OndoREui is the public-facing Next.js site for Ondo Real Estate. It serves:

- **Marketing & lead generation** — homepage, service pages, about, contact
- **Property search** — listings with map view (Leaflet)
- **Financial calculators** — mortgage, affordability, ROI, cap rate, DSCR, and more
- **Investment opportunities** — curated deals with due-diligence data
- **Neighborhood and school guides** — local market intelligence for Utah cities
- **Market reports** — Wasatch Front data and analysis
- **Blog** — real estate and technology editorial
- **Tenant/Owner portals** — logged-in dashboards (linked to OndoREDashboard)
- **Onboarding flows** — invite acceptance, application, visit confirmation

## Tech Stack

- **Framework**: Next.js 15 App Router (TypeScript)
- **Styling**: Tailwind CSS with design tokens (`src/styles/design-tokens.json`)
- **Components**: shadcn/ui, Radix UI
- **Database/Auth**: Supabase JS client (`NEXT_PUBLIC_SUPABASE_URL`)
- **Payments**: Stripe
- **Maps**: Leaflet (property search map)
- **Testing**: Vitest, React Testing Library, axe (accessibility)
- **SEO**: `lib/seo.ts`, `lib/site.ts`, JSON-LD structured data, `next-sitemap`
- **Analytics**: `lib/analytics.ts`, `lib/supabase-analytics.ts`
- **i18n**: 8 locales — en, es, fr, it, te, hi, ta, kn (locale files in `public/locales/`)
- **AI discovery**: `/llms.txt`, `/llms-full.txt`, `/llms.json` (dynamic Next.js routes)
